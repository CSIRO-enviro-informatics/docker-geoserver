docker-geoserver
================

Docker deployment of Geoserver. Currently this includes:
* Postgres 9.3 / Postgis 2.1
* Tomcat 7.0.55
* Geoserver 2.5.3 with App Schema plugin

Quick-start
----------
Build:
   sudo ./build.sh


Run:
   sudo ./run.sh

Clean-up:
   sudo ./clean-up.sh

Inspect using `docker ps`
